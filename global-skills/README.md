# Global Skills

These skills are intended to be copied to your global Windsurf skills directory so they're available in all projects.

## Installation

Copy these folders to your global skills directory:

**Windows:**
```
%USERPROFILE%\.codeium\windsurf\skills\
```

**macOS/Linux:**
```
~/.codeium/windsurf/skills/
```

## Available Global Skills

| Skill | Description |
|-------|-------------|
| `ai-research-tutor` | Systematic research methodology for AI when investigating new topics |
| `python-best-practices` | Python coding standards, type hints, async patterns |
| `twelve-factor-app` | 12-factor methodology for cloud-native applications |
| `testing-pytest` | pytest patterns, fixtures, async testing, mocking |

## Usage

After copying, these skills will be automatically available in all your Windsurf projects.

Invoke manually with `@skill-name` or let Cascade auto-invoke based on context.
